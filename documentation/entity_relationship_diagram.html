Diseño de Datos (Diagrama Entidad-Relación)

Definición de las entidades principales del sistema y sus relaciones.

Entidades clave:

  • Alumno: ID, Nombre, Apellido, FechaNacimiento, Dirección, ContactoEmergencia.
  • Padre/Tutor: ID, Nombre, Apellido, RelacionConAlumno, Teléfono, Email, Dirección.
  • Personal: ID, Nombre, Apellido, Rol, Teléfono, Email, Dirección.
  • Dirección: ID, Calle, Número, Ciudad, Provincia, CódigoPostal.

Relaciones:

  • Un Alumno puede tener varios Padres/Tutores.
  • Un Padre/Tutor puede tener varios Alumnos.
  • Un Alumno, Padre/Tutor o Personal puede tener una o varias Direcciones.
